# Arquitectura del Portal Corporativo

Plataforma corporativa agnóstica al giro de negocio.

## Componentes

- Portal Angular
- API Gateway
- Security API
- Menu API
- Integration API
- Reporting API
- Audit API
- Notification API
- Content API
- Master Catalog API
- Configuration API

## Modelo base

La solución se modela con elementos genéricos:

- Sistemas
- Aplicaciones
- Grupos
- Recursos
- Permisos
- Perfiles
- Catálogos
- Formularios
- Reportes
- Contenidos
- Integraciones
- Metadata configurable
