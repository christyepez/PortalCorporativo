# Gobierno de datos / MODA

La solución debe manejar gobierno de datos desde el diseño:

- Data ownership.
- Data stewardship.
- Clasificación de datos.
- Metadata.
- Linaje.
- Calidad.
- Contratos de datos.
- Auditoría.
- Retención.

## Clasificación sugerida

- PUBLIC
- INTERNAL
- CONFIDENTIAL
- RESTRICTED
- SENSITIVE
- SECRET

## Regla

Cada dominio es dueño de sus datos. La integración entre dominios se realiza por APIs o eventos.
