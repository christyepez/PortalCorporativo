# Decisiones de stack tecnológico

Backend: .NET 8 / ASP.NET Core.

Frontend: Angular.

Contenedores: Docker Compose.

Base de datos: SQL Server con bases por dominio lógico.

Eventos: SQL Outbox en la primera etapa. Broker dedicado preparado para una fase futura.

Cache: Redis.

Contenido: proveedor local para desarrollo y proveedores empresariales configurables.

Logs: Seq.
